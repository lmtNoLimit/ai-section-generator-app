# Budget Allocation Framework

## Allocation Models

### 70-20-10 Rule
- **70%** Proven channels (best ROI)
- **20%** Growth channels (scaling)
- **10%** Experimental (testing new)

### Objective-Based
| Objective | Channel Focus |
|-----------|---------------|
| Awareness | Display, Video, Social |
| Consideration | Search, Content, Retargeting |
| Conversion | Search, Retargeting, Email |
| Retention | Email, CRM, Loyalty |

### Funnel-Based
- **Top of Funnel:** 30-40%
- **Middle of Funnel:** 30-40%
- **Bottom of Funnel:** 20-30%

## Channel Benchmarks

| Channel | Typical % | When to Increase |
|---------|-----------|------------------|
| Paid Search | 25-35% | High-intent keywords available |
| Paid Social | 20-30% | Good audience match |
| Display | 10-15% | Awareness goals |
| Email | 5-10% | Good list size |
| Content | 10-20% | SEO opportunity |
| Influencer | 5-15% | Authentic fit |

## Budget Pacing

### Daily Budget
```
Daily Budget = Total Budget / Campaign Days
```

### Pacing Strategies
- **Even:** Same spend daily
- **Frontloaded:** Higher early, optimize later
- **Backloaded:** Start slow, increase proven
- **Dayparted:** Vary by day/time

### Pacing Check
```
Expected Spend = (Days Elapsed / Total Days) Ã— Budget
Variance = Actual Spend - Expected Spend
```

## Reallocation Triggers

**Increase Budget When:**
- ROAS > target by 20%+
- CVR significantly above average
- Plenty of impression share headroom

**Decrease Budget When:**
- ROAS < target for 3+ days
- CVR declining trend
- Quality score dropping

**Move Budget When:**
- One channel outperforming another
- Audience saturation reached
- New opportunity identified
