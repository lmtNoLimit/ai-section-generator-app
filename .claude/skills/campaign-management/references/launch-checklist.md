# Campaign Launch Checklist

## Pre-Launch (1 Week Before)

### Strategy
- [ ] Campaign brief approved
- [ ] Budget allocated and approved
- [ ] Timeline confirmed
- [ ] Roles and responsibilities assigned
- [ ] Stakeholder communication plan set

### Creative Assets
- [ ] All ad creatives complete and approved
- [ ] Landing pages live and tested
- [ ] Email templates approved
- [ ] Social posts scheduled
- [ ] Copy variations ready for A/B testing

### Technical Setup
- [ ] Tracking pixels installed
- [ ] UTM parameters defined
- [ ] Conversion events configured
- [ ] Attribution model set
- [ ] Reporting dashboard ready

### Platform Setup
- [ ] Ad accounts funded
- [ ] Campaigns built in platforms
- [ ] Audiences created and verified
- [ ] Bidding strategy selected
- [ ] Daily budgets configured

## Launch Day

### Pre-Launch (Morning)
- [ ] Final QA of all links
- [ ] Test conversions tracking
- [ ] Confirm all assets approved
- [ ] Team notified of launch time
- [ ] Monitoring plan in place

### Launch
- [ ] Campaigns activated
- [ ] First impressions/clicks verified
- [ ] No errors in platforms
- [ ] Landing pages loading properly
- [ ] Slack/email announcement sent

### Post-Launch (Same Day)
- [ ] Verify data flowing to analytics
- [ ] Check early performance metrics
- [ ] Address any immediate issues
- [ ] Update stakeholders

## Post-Launch (48-72 Hours)

- [ ] Review initial performance
- [ ] Pause underperformers
- [ ] Scale winners if applicable
- [ ] Document early learnings
- [ ] Adjust bidding if needed
- [ ] Plan first optimization round

## Weekly During Campaign

- [ ] Performance review
- [ ] Budget pacing check
- [ ] Optimization actions
- [ ] Stakeholder update
- [ ] Competitive monitoring
