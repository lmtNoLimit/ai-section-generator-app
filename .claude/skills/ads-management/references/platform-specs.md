# Ad Platform Specifications

## Google Ads

### Ad Types
| Type | Best For | Specs |
|------|----------|-------|
| Search | High intent | 3 headlines (30 chars), 2 descriptions (90 chars) |
| Display | Awareness | 1200x628, 300x250, 336x280 |
| Video | Engagement | 15-30 sec, 16:9 |
| Performance Max | Full funnel | Multiple assets |

### Bidding Strategies
- Manual CPC: Control bids
- Target CPA: Optimize conversions
- Target ROAS: Optimize value
- Maximize Conversions: Volume focus

## Meta Ads (Facebook/Instagram)

### Ad Types
| Type | Placement | Specs |
|------|-----------|-------|
| Image | Feed/Stories | 1080x1080, 1080x1920 |
| Video | Feed/Reels | 1:1/9:16, 15-60 sec |
| Carousel | Feed | 2-10 cards |
| Collection | Mobile | Hero + products |

### Objectives
- Awareness: Reach, Brand awareness
- Traffic: Link clicks, Landing page views
- Engagement: Post engagement, Messages
- Conversions: Purchases, Leads

## LinkedIn Ads

### Ad Types
| Type | Best For | Specs |
|------|----------|-------|
| Sponsored Content | Awareness | 1200x627 |
| Message Ads | Direct response | 500 char body |
| Text Ads | Traffic | 25 char headline |
| Dynamic Ads | Personalization | Auto-generated |

### Targeting
- Job title, Function, Seniority
- Company size, Industry
- Skills, Groups, Interests
