# Ads Optimization Playbook

## Daily Checks

- [ ] Spend on pace?
- [ ] Any anomalies in metrics?
- [ ] Disapproved ads?
- [ ] Budget depleted early?

## Weekly Optimization

### Performance Review
| Metric | Action if Low |
|--------|---------------|
| CTR < 1% | Test new creative/copy |
| CVR < 2% | Check landing page |
| CPC too high | Refine targeting |
| ROAS < 2x | Pause or restructure |

### Optimization Actions
1. Pause ads with 0 conversions (100+ clicks)
2. Increase budget on ROAS > target
3. Add negative keywords (search)
4. Refresh creative (fatigue check)
5. Test new audiences

## A/B Testing Framework

### What to Test
1. **Creative** (highest impact)
2. **Audience** (high impact)
3. **Copy** (medium impact)
4. **Bidding** (medium impact)
5. **Placements** (lower impact)

### Testing Rules
- One variable per test
- Minimum 100 conversions per variant
- Run 7-14 days minimum
- 95% confidence before deciding
- Document all tests

## Scaling Strategy

### Vertical Scaling
- Increase budget 20-30% at a time
- Wait 3-5 days between increases
- Monitor CPA/ROAS changes

### Horizontal Scaling
- Duplicate winning ad sets
- Test new audiences
- Expand to new platforms
- Create variations of winners

## Troubleshooting

### High CPC
- Narrow targeting
- Improve Quality Score
- Test different placements
- Adjust bid strategy

### Low CTR
- Test new headlines
- Refresh creative
- Check audience fit
- Review ad relevance

### Low ROAS
- Exclude low-value audiences
- Focus on bottom-funnel
- Test higher-value products
- Review attribution window

## Budget Allocation

| Performance | Action |
|-------------|--------|
| ROAS > 4x | Scale aggressively |
| ROAS 2-4x | Scale gradually |
| ROAS 1-2x | Optimize first |
| ROAS < 1x | Pause or restructure |
