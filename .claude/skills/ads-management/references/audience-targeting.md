# Audience Targeting Guide

## Targeting Types

### 1. Demographic
- Age, Gender, Location
- Income, Education
- Parental status
- Home ownership

### 2. Interest-Based
- Topics and categories
- Page likes/follows
- Content engagement
- Purchase behavior

### 3. Behavioral
- Purchase history
- Device usage
- Travel patterns
- Life events

### 4. Custom Audiences
- Website visitors (pixel)
- Customer lists (email/phone)
- App users
- Video viewers
- Engagement (social)

### 5. Lookalike/Similar
- Based on custom audiences
- 1% = most similar
- 1-10% range available

## Platform-Specific Targeting

### Google Ads
- Keywords (search intent)
- In-market audiences
- Affinity audiences
- Remarketing lists
- Customer Match

### Meta Ads
- Detailed targeting (interests)
- Custom audiences
- Lookalike audiences
- Advantage+ (AI)

### LinkedIn
- Job title, Function
- Company, Industry
- Skills, Groups
- Seniority level

### TikTok
- Interest categories
- Behavioral targeting
- Custom/Lookalike
- Creator audiences

## Audience Strategy

### Funnel-Based Targeting
```
TOFU (Awareness)
├── Broad interests
├── Lookalike 5-10%
└── In-market audiences

MOFU (Consideration)
├── Engaged visitors
├── Lookalike 1-5%
└── Specific interests

BOFU (Conversion)
├── Website visitors
├── Cart abandoners
├── Customer lookalike 1%
└── Email lists
```

## Exclusions (Critical)

- Existing customers (if acquisition)
- Converted users
- Irrelevant locations
- Competitors' employees
- Low-quality placements

## Testing Approach

1. Start with 2-3 audiences
2. Same creative across audiences
3. Sufficient budget per audience
4. Run 7-14 days minimum
5. Scale winner, pause losers
