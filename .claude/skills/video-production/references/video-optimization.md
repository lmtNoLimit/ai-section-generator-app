# Video Optimization Guide

## Thumbnail Best Practices

### Design Rules
- Bright, high-contrast colors
- Face with expression (if applicable)
- Large, readable text (3-4 words max)
- Consistent brand style
- Test at small size (mobile)

### Thumbnail Specs
| Platform | Size | Safe Zone |
|----------|------|-----------|
| YouTube | 1280x720 | Center 60% |
| TikTok | 1080x1920 | Top third |
| Instagram | 1080x1080 | Center |

### What Works
- Curiosity gap (incomplete info)
- Before/after contrast
- Numbers and lists
- Emotion (surprise, curiosity)

## Title Optimization

### YouTube Title Formula
`[Keyword] - [Benefit/Outcome] ([Qualifier])`

Examples:
- "Email Marketing - 5 Templates That Convert (Free Download)"
- "Learn Python - Build Your First App in 30 Minutes"

### Short-Form Captions
- Hook in first line
- Add context/story
- Include hashtags
- CTA at end

## SEO for Video

### YouTube SEO
1. Keyword in title (front-load)
2. Keyword in description (first 150 chars)
3. Tags (primary + related)
4. Chapters with keywords
5. Closed captions

### Description Template
```
[Hook - 1-2 sentences with keyword]

[Timestamps/Chapters]
0:00 Intro
1:30 [Topic 1]
...

[Links and Resources]

[About]

[Hashtags]
```

## Retention Optimization

### Improve Watch Time
- Strong hook (first 5 sec)
- Pattern interrupts every 30 sec
- Open loops (tease what's coming)
- Visual variety (cuts, b-roll)
- Remove dead air

### Reduce Drop-off
- Cut slow intros
- Front-load value
- Use chapters
- Match thumbnail promise

## Repurposing Strategy

### Long → Short
- Extract 3-5 best moments
- Add captions
- Adjust aspect ratio
- Platform-specific hooks

### One Video → Multiple Assets
- YouTube long-form
- YouTube Shorts (3-5 clips)
- TikTok/Reels versions
- LinkedIn cut
- Blog post embed
- Email thumbnail + link
- Social media clips
