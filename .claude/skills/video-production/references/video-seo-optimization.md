# Video SEO Optimization Guide

Maximize video discoverability across platforms through SEO best practices.

## Platform SEO Overview

| Platform | Primary Discovery | Algorithm Factors |
|----------|------------------|-------------------|
| YouTube | Search + Suggested | Watch time, CTR, engagement |
| TikTok | For You Page | Completion rate, shares, saves |
| Instagram | Explore + Reels | Engagement velocity, saves |
| LinkedIn | Feed + Search | Professional relevance, engagement |

## YouTube SEO

### Title Optimization

**Structure:**
```
[Primary Keyword] - [Benefit/Hook] ([Qualifier])
```

**Examples:**
```
Email Marketing Tutorial - 5 Templates That Convert (Free Download)
Python for Beginners - Build Your First App in 30 Minutes
SEO Strategy 2025 - Complete Guide for Ranking #1
```

**Best Practices:**
- Primary keyword at start (first 40 characters)
- Include benefit or outcome
- Add qualifiers: (Tutorial), (Free), (2025)
- 50-60 characters ideal
- Don't exceed 100 characters

### Description Strategy

**Template:**
```markdown
{Hook with primary keyword - 150 characters}

{2-3 sentence value proposition}

TIMESTAMPS:
0:00 Intro
1:30 {Topic with keyword}
3:45 {Topic with keyword}
...

RESOURCES:
{Links to mentioned resources}
{Your products/services}

RELATED VIDEOS:
{Links to your other videos}

ABOUT:
{Brief channel description}

TAGS:
#{keyword1} #{keyword2} #{keyword3}
```

**Key Points:**
- First 150 characters most critical (shows in search)
- Include primary keyword in first sentence
- Add timestamps with keywords
- Link to resources, products, related content
- 2000-5000 characters ideal

### Tags

**Hierarchy:**
1. Exact match keyword
2. Broad keyword variations
3. Related topics
4. Channel series tags
5. Common misspellings

**Example for "Email Marketing Tutorial":**
```
email marketing tutorial
email marketing
email marketing for beginners
how to write marketing emails
email copywriting
email templates
marketing tutorials
digital marketing
```

### Chapters (Timestamps)

**Benefits:**
- Improves user experience
- Creates rich snippets in search
- Allows keyword-rich section titles

**Format:**
```
0:00 Introduction to Email Marketing
1:24 Setting Up Your Email Tool
3:15 Writing Subject Lines That Convert
5:42 Email Body Best Practices
8:30 Testing and Optimization
```

### Engagement Signals

**Boost These:**
- Watch time (keep viewers watching)
- Click-through rate (compelling thumbnail + title)
- Likes, comments, shares
- Subscriber actions
- Playlist additions

**In-Video CTA Strategy:**
```
[0:30] "Hit subscribe if you want more {topic}"
[Middle] "Comment below with your {question}"
[End] "Watch this next: {related video}"
```

## TikTok SEO

### Discoverability Factors

1. **Hashtags** - Primary discovery mechanism
2. **Captions** - Keywords for search
3. **Audio** - Trending sounds boost reach
4. **Engagement** - Early engagement crucial

### Hashtag Strategy

**Structure:**
```
#NicheSpecific #BroadReach #Trending #BrandedHashtag
```

**Categories:**
- Niche: #EmailMarketingTips (smaller, targeted)
- Broad: #MarketingTips (larger audience)
- Trending: #FYP #ForYou #Viral (discovery)
- Branded: #YourBrandName (building presence)

**Optimal Count:** 3-5 hashtags

### Caption Format

```
Hook line that grabs attention

{Brief description with keywords}

#hashtag1 #hashtag2 #hashtag3
```

### Audio/Sound Strategy

- Use trending sounds when relevant
- Original audio for tutorials
- Mix trending + original strategically
- Save sounds to use later

## Instagram Reels SEO

### Caption Strategy

**Structure:**
```
Hook line {emoji}

Value statement with keywords

CTA: Save this for later!

#hashtag1 #hashtag2 ... (up to 30, use 10-15)
```

### Hashtag Research

**Categories:**
- Industry: #DigitalMarketing (1M+ posts)
- Specific: #EmailMarketingTips (10K-100K)
- Niche: #SaaSMarketing (1K-10K)
- Location: #MarketingNYC (if relevant)

### Cover Image/Thumbnail

- First frame matters
- Can upload custom cover
- Include text overlay
- Match feed aesthetic

## LinkedIn Video SEO

### Native Video Benefits

- Algorithm favors native uploads
- 5x more engagement than shared links
- Professional audience targeting

### Caption Optimization

**Structure:**
```
{Strong opening line}

{Problem statement}

{3-5 key points}

{CTA with question}

#hashtag #hashtag #hashtag (3-5 max)
```

### Professional Keywords

Focus on:
- Industry terms
- Job titles
- Skill keywords
- Solution language

## Cross-Platform Strategy

### Content Adaptation

| Element | YouTube | TikTok | Reels | LinkedIn |
|---------|---------|--------|-------|----------|
| Aspect | 16:9 | 9:16 | 9:16 | 16:9/1:1 |
| Length | 8-15min | 15-60s | 15-30s | 30s-2min |
| Captions | Optional | Essential | Essential | Essential |
| Hashtags | Tags | Caption | Caption | Caption |
| Tone | Informative | Trendy | Lifestyle | Professional |

### Repurposing Flow

```
Long-form YouTube Video
        ↓
Extract 3-5 Best Moments
        ↓
┌─────────────────────────────────┐
│  TikTok  │  Reels  │  LinkedIn  │
│  Clips   │  Clips  │  Clip      │
└─────────────────────────────────┘
        ↓
Adapt captions/hashtags per platform
```

## Keyword Research

### YouTube Keywords

**Tools:**
- YouTube Search Suggest
- TubeBuddy / VidIQ
- Google Trends (YouTube search)
- Competitor analysis

**Process:**
1. Start typing in YouTube search
2. Note autocomplete suggestions
3. Check "People also watched"
4. Analyze top-ranking videos
5. Use research tools for volume

### TikTok Keywords

**Tools:**
- TikTok Search
- Trend Discovery
- Creative Center
- Hashtag analytics

**Process:**
1. Search your topic
2. Check trending hashtags
3. Analyze viral videos in niche
4. Note common phrases

### Cross-Platform Research

1. **Search volume:** How often searched?
2. **Competition:** How hard to rank?
3. **Intent:** What does searcher want?
4. **Trend:** Rising or declining?

## Technical SEO

### File Naming

```
Good: email-marketing-tutorial-2025.mp4
Bad: video_final_v3.mp4
```

### Closed Captions

**Benefits:**
- Accessibility
- SEO value (YouTube reads captions)
- Sound-off viewing
- Translation support

**Implementation:**
- Auto-generated (edit for accuracy)
- Upload SRT file
- Add burned-in captions

## Analytics & Iteration

### Key Metrics

| Metric | Target | Action if Low |
|--------|--------|---------------|
| CTR | >5% | Improve thumbnail/title |
| Watch Time | >50% | Better hooks, pacing |
| Engagement | Increasing | Add more CTAs |
| Search Rank | Top 10 | Optimize keywords |

### A/B Testing

**Test:**
- Thumbnails (biggest impact)
- Titles (keyword placement)
- First 30 seconds
- CTA placement

### Optimization Loop

```
Create → Publish → Analyze (72h)
           ↓
    Low performance?
           ↓
Update thumbnail/title → Monitor
           ↓
Still low? → Analyze competition
           ↓
Create improved version
```

## Quick Checklists

### Pre-Publish

- [ ] Primary keyword in title (first 40 chars)
- [ ] Description has keyword in first sentence
- [ ] Timestamps with keywords
- [ ] 5-10 relevant tags (YouTube)
- [ ] Thumbnail A/B ready
- [ ] Captions added/verified
- [ ] Cards and end screens set
- [ ] Playlist assignment

### Post-Publish (24-72h)

- [ ] Check CTR (update thumbnail if <4%)
- [ ] Review retention graph
- [ ] Respond to comments
- [ ] Share to other platforms
- [ ] Analyze search impressions
- [ ] Compare to similar videos

### Monthly Review

- [ ] Top performing videos analysis
- [ ] Keyword ranking changes
- [ ] Competitor new content review
- [ ] Trending topics check
- [ ] Content gap identification
- [ ] Update older videos
